# おじさんインベーダー 🎮

レトロスタイルのブラウザゲーム。中年男性がネクタイビームでキャバ嬢を倒し、¥500,000札を避けるシューティングゲームです。

## 🎯 ゲーム概要

- **プレイヤー**: 中年男性（ネクタイを武器として使用）
- **敵**: 降下してくるキャバ嬢
- **回避対象**: ¥500,000札（触れるとゲームオーバー）
- **操作**: 矢印キーで移動、スペースキーで攻撃
- **目標**: キャバ嬢を倒してスコアを稼ぎ、お金を避ける

## 🏗️ アーキテクチャ

### システム構成
```
おじさんインベーダー/
├── index.html              # メインHTMLファイル
├── js/
│   ├── game-engine.js      # ゲームエンジン（コア機能）
│   ├── game-objects.js     # ゲームオブジェクト（プレイヤー、敵、弾丸等）
│   ├── input-system.js     # 入力システム（キーボード処理）
│   ├── ui-system.js        # UIシステム（スコア、ライフ表示）
│   ├── sound-system.js     # サウンドシステム（Web Audio API）
│   └── main.js             # メインゲームファイル（統合）
├── test/
│   ├── unit-tests.js       # ユニットテスト
│   ├── e2e-tests.js        # E2Eテスト
│   └── test-runner.html    # テストランナー
└── README.md               # プロジェクトドキュメント
```

### 技術スタック
- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **グラフィックス**: HTML5 Canvas API
- **音響**: Web Audio API
- **テスト**: カスタムテストフレームワーク
- **アーキテクチャ**: モジュラー設計、コンポーネント分離

## 🎮 ゲーム機能

### コア機能
1. **ゲームループ**: 60FPSで動作する効率的なゲームループ
2. **衝突検出**: 正確な矩形衝突判定システム
3. **オブジェクト管理**: 動的なゲームオブジェクトの生成・削除
4. **状態管理**: ゲーム状態（開始、プレイ中、ゲームオーバー）の管理

### ゲームオブジェクト
- **Player**: 中年男性キャラクター（移動、攻撃）
- **Bullet**: ネクタイビーム（上方向に移動）
- **Enemy**: キャバ嬢（下方向に移動、アニメーション）
- **Money**: ¥500,000札（回転しながら下方向に移動）
- **Explosion**: 爆発エフェクト（パーティクルシステム）

### 入力システム
- **移動**: 左右矢印キー
- **攻撃**: スペースキー、上矢印キー
- **ゲーム制御**: スペースキー（開始・リスタート）

### サウンドシステム
- **効果音**: ビーム発射、爆発、ゲームオーバー、レベルアップ
- **BGM**: シンプルなループ音楽
- **音量制御**: マスターボリューム調整

## 🚀 実行方法

### 前提条件
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- Web Audio API対応
- JavaScript有効

### 起動方法
1. プロジェクトをクローン
```bash
git clone <repository-url>
cd ojisan-invader-app
```

2. ローカルサーバーを起動（推奨）
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

3. ブラウザでアクセス
```
http://localhost:8000
```

### 直接実行
`index.html`をブラウザで直接開くことも可能ですが、一部の機能（音響等）が制限される場合があります。

## 🎯 ゲームプレイ

### 操作方法
- **←→**: 左右移動
- **スペース**: ネクタイビーム発射
- **↑**: ネクタイビーム発射（代替）

### ゲームルール
1. キャバ嬢を倒すと100点獲得
2. キャバ嬢に触れるとライフ-1
3. ¥500,000札に触れるとゲームオーバー
4. 1000点ごとにレベルアップ（敵の速度上昇）
5. ライフが0になるとゲームオーバー

### スコアシステム
- キャバ嬢撃破: +100点
- レベルアップ: 1000点ごと
- ハイスコア: ブラウザのローカルストレージに保存
